# Neuralab

Neuralab is a cutting-edge discovery and mapping platform that turns **viral social media trends** into actionable insights for **Solana ecosystem tokens**. By leveraging social signals from platforms like **TikTok, Instagram Reels, and X (formerly Twitter)**, Neuralab helps users find and explore Solana coins tied to viral memes, cultural moments, and trending content.

### Prerequisites

To use Neuralab, you only need:

- A modern web browser (Chrome, Firefox, Safari, etc.)
- A valid social media link or an image for trend input

### Usage

1. Visit the Neuralab web app: `https://neuralab.dev/`  
2. Paste a **TikTok / Instagram Reel / X link** into the search input  
3. Or upload an image related to the trend  
4. Or describe the trend in text  
5. Click **Find Coin**  
6. Neuralab will return results if any Solana coins match the trend
